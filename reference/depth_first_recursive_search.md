# Layouts for snapping layouts to a grid

The function uses approximate pattern matching to redistribute coarse
layouts on square grid points, while preserving the topological
relationships among the nodes (see Inoue et al. 2012).

## Usage

``` r
depth_first_recursive_search(layout)
```

## References

Inoue, Kentaro, Shinichi Shimozono, Hideaki Yoshida, and Hiroyuki
Kurata. 2012. “Application of Approximate Pattern Matching in Two
Dimensional Spaces to Grid Layout for Biochemical Network Maps” edited
by J. Bourdon. *PLoS ONE* 7(6):e37739.
