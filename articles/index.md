# Articles

### Tutorials

Static, read-only versions of the interactive `learnr` tutorials bundled
with the package. Install the package and run `run_tute()` for the full
interactive experience.

  - [Visualising
    Networks](https://stocnet.github.io/autograph/articles/visualising-networks.md):
    
    This tutorial covers how to visualise networks and network-related
    results, from a first automatic graph with graphr() through mapping
    attributes to colour, shape, and size, choosing themes and layouts,
    arranging and animating multiple graphs, and exporting
    publication-ready figures.
